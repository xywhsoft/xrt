

# xrt Project Documentation

## Project Overview
xrt is a multifunctional C library that provides a wide range of capabilities, including memory operations, character set conversion, file handling, hash computation, mathematical operations, operating system interfaces, path handling, string manipulation, and time handling. This library is designed to streamline the development process and deliver efficient cross-platform support.

## Key Features
- **Memory Operations**: Provides memory management functions such as `xrtMalloc`, `xrtCalloc`, `xrtRealloc`, and `xrtFree`.
- **Character Set Conversion**: Supports conversion between UTF-8, UTF-16, and UTF-32, including functions like `xrtUTF8to16`, `xrtUTF16to8`, etc.
- **Hash Computation**: Offers 32-bit and 64-bit hash functions, such as `xrtHash32`, `xrtHash64`, etc.
