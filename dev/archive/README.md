# XRT Archive Root

This directory stores superseded or transitional foundation code that must no
longer remain in mainline.

Archive policy:

- archived code is not part of the production build
- archived code is not part of the public mainline API
- every archived subtree should explain what was removed and why

Subdirectories:

- `runtime/`
- `coroutine/`
- `xnet/`
- `app/`
