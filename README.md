

# xrt 项目说明文档

## 项目简介
xrt 是一个多功能的 C 语言库，提供了包括内存操作、字符集转换、文件处理、哈希计算、数学运算、操作系统接口、路径处理、字符串处理和时间处理在内的多种功能。该库旨在简化开发流程，提供高效的跨平台支持。

## 主要功能
- **内存操作**：提供 `xrtMalloc`, `xrtCalloc`, `xrtRealloc`, `xrtFree` 等内存管理函数。
- **字符集转换**：支持 UTF-8, UTF-16, UTF-32 之间的转换，如 `xrtUTF8to16`, `xrtUTF16to8` 等。
- **哈希计算**：提供 32 位和 64 位的哈希函数，如 `xrtHash32`, `xrtHash64` 等。
